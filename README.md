## Descripción

La matemática constituye uno de los pilares fundamentales en la formación de un **Ingeniero de Audio**. Los conceptos desarrollados en este curso proporcionan las herramientas necesarias para comprender y modelar fenómenos presentes en la acústica, el procesamiento digital de señales (DSP), la electrónica, el análisis de sistemas y el diseño de equipos de audio.

El objetivo de este repositorio es recopilar el material del curso **Fundamentos de Matemática para Ingeniería de Audio**, ofreciendo las presentaciones utilizadas en clase como material de consulta para estudiantes y docentes.

## Objetivos de aprendizaje

Al finalizar este curso, el estudiante será capaz de:

- Comprender el uso de sistemas de unidades y realizar conversiones.
- Aplicar correctamente la jerarquía de operaciones.
- Resolver operaciones con fracciones.
- Analizar relaciones y funciones matemáticas.
- Representar y operar vectores en diferentes sistemas de coordenadas.
- Resolver sistemas de ecuaciones lineales utilizando distintos métodos matriciales.
